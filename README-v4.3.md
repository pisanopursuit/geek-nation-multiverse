# Geek Nation Multiverse — Universe Engine V4.3

Version 4.3 completes the user-facing Universe concept for class demonstrations.

## Included

- Unlimited nested universes using the existing `parent_id` hierarchy.
- Clear hierarchical parent selection in Universe Admin.
- “Add Child” actions in the admin list.
- “Add Sub-Universe” actions on public universe profiles for administrators and moderators.
- Empty-state prompt for creating the first world inside a universe.
- A community Billboard on every universe profile.
- Root conversations and threaded replies.
- Join-before-posting permission.
- Public reading, authenticated participation, and moderator hide controls.

## Install

Upload the files over Version 4.2, sign in as an administrator, and visit:

`upgrade-universe-billboards.php`

Click **Install Upgrade**. The upgrade is safe to run more than once.
