# Geek Nation Multiverse — Universe Community V4.4

## Added
- Billboard post deletion by the post/reply author.
- Administrator and universe moderator deletion of any billboard post or reply.
- Optional links and image attachments on billboard posts.
- Live chat for every universe.
- Sidebar chat card that can pop out into a floating chat window.
- Chat message deletion by the message author, administrator, or universe moderator.

## Install
Upload this package over Version 4.3 and run:

`upgrade-universe-community.php`

The upgrade is safe to run more than once.
