# Geek Nation Multiverse Prototype

A responsive front-end prototype for a permanent online geek convention.

## Included
- Immersive homepage using the selected Geek Nation Multiverse logo
- Universe browsing
- Filterable virtual booths
- Panels and event schedule
- Artist Alley
- Education / Multiverse Academy
- Buy, sell, and trade marketplace
- Sign-in and booth application modal prototypes
- Responsive mobile navigation

## Run locally
Open `index.html` in a browser.

## Production next steps
Connect authentication, profiles, booth management, product inventory, payments, livestreams, search, messaging, event schedules, and an admin dashboard to a back end.
