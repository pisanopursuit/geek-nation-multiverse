# Geek Nation Multiverse Version 5 — Booths & Marketplace Foundation

Upload the package over Version 4.4, sign in as an administrator, and run `upgrade-booths-marketplace.php`.

Version 5 adds exhibitor booths, public booth profiles, company/brand/creator ownership, universe assignments, product catalogs, convention tags, session-based shopping cart, demo checkout, seller order management, inventory reduction, and administrator booth approvals.

No live payment is processed. Orders use the `demo` provider with pending payment status. Stripe, PayPal, Square, or another gateway can be connected later without rebuilding products, carts, or orders.
