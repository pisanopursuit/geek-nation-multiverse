# Geek Nation Multiverse — Version 5.1 Booth Management

Version 5.1 completes the booth-owner management workflow while leaving live payments disabled.

## Added

- Tabbed booth management dashboard
- Overview metrics and completion checklist
- Public booth status, location, hours, support email, and social links
- Product creation, editing, inventory, tags, and archiving
- Full order details with order and payment-status controls
- Booth team management with manager, staff, artist, and moderator roles
- Booth gallery
- Public file downloads and external resources
- Shipping and return policies
- Daily booth-view analytics
- Expanded public booth page

## Install

Upload the Version 5.1 files over Version 5, sign in as an administrator, and visit:

`upgrade-booth-management.php`

Click **Install Version 5.1** once. Existing booths, products, carts, and orders are preserved.

## Access rules

- Booth owners and administrators have full booth access.
- Managers can edit and operate the booth.
- Staff, artist, and moderator roles are recorded for public team display and future permissions.
- Only the booth owner or an administrator can add or remove team members.
